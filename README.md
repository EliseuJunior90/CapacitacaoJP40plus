
# Sistema de Gerenciamento de Itens

Aplicação backend desenvolvida em Java com Spring Boot para gerenciamento de itens e categorias, com persistência em banco de dados Oracle.
O sistema permite operações completas de CRUD, garantindo integridade dos dados e regras de negócio.


## Funcionalidades

- Cadastro de itens
- Atualização de itens
- Remoção de itens
- Listagem de itens
- Cadastro de categorias
- Associação de itens com categorias
- Validação de dados (ex: preço maior que zero)
- Prevenção de categorias duplicadas
